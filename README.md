# Inngangur-ad-maltaekni

## Verkefni

**hw1**: Vinna með tokenization, þar sem ég breyti texta í tokens og vinn með mismunandi aðferðir við að brjóta niður texta.

**hw2**: Lemmatization, ngram extraction, og parser fyrir noun phrases og verb phrases. Verkefnið snýst um að greina hluta setninga og vinna með orðaform.

**hw3**: Vinna með pretrained word embeddings (word2vec og GloVe) til að finna hversu svipuð orð eru að merkingu, prófa analogies, og nota masked language model BERT til að spá fyrir um falin orð.

**hw4**: Prófa pretrained models fyrir question answering, named entity recognition, og translation með íslensku. Geri einnig einfalt user interface fyrir modelin

**hw5**: Named entity recognition með regex patterns og spaCy, búa til Wiki links fyrir entities, og extrakta relations á milli entities í texta
